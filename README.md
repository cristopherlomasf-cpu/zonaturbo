# 🏎️ Zona Turbo

Aplicación móvil del emprendimiento **Zona Turbo** — barbería y estética para caballeros ubicada en Tulcán, Carchi, Ecuador.

---

## 📋 Descripción

**Zona Turbo** es una aplicación desarrollada en Kotlin para Android que permite a los clientes interactuar con los servicios del local, consultar información y gestionar su experiencia de manera sencilla y rápida.

## 🚀 Características

- Información del negocio y servicios disponibles
- Contacto directo con el local
- Diseño moderno y adaptado al estilo de la marca

## 🛠️ Tecnologías

| Tecnología | Uso |
|---|---|
| Kotlin | Lenguaje principal |
| Android Studio | Entorno de desarrollo |
| XML | Diseño de interfaces |

## ⚙️ Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/cristopherlomasf-cpu/zonaturbo.git
   ```
2. Abre el proyecto en **Android Studio**.
3. Sincroniza las dependencias con Gradle.
4. Ejecuta la aplicación en un emulador o dispositivo físico.
